Cloning

* If you are cloning somebody else's repo, Fork it first
  1. Copy the URL for the new repo
  2. Open Git Bash in your working directory
  3. $ git clone https://github.com/CCSudbury/GitTogether

Initializing

  1. Open the working directory
  2. Write your first bit of code
  3. git init; git add . ; git commit
  4. Connect to GitHub
    1. Create a new repo
    2. Push an existing repository
    3. $ git remote add origin https://github.com/username/new_repo $ git push -u origin master

How Commits Work

  • Each commit has a unique ID of 40 characters (SHA-1 Hash) containing information about the commit such as the parent commit

2-step Commit

Git has three sections:
  1. Working Directory
  2. Staging Area
  3. History
git add moves from PWD to Staging git commit moves from Staging to History
Example:
  • Working on a website you edit a few pages, but you also add some missing images from another page.
  • You can add and commit them independently to highlight that the changes were not related

Merging

  • Let's bring those changes back to the project
  • Use git merge TARGET_BRANCH_NAME to merge the current branch's changes with the target
  • Ex: git merge remote/master to merge with remote's master.

Conflicts

  • git status indicates which files have conflicting changes between branches of the merge
  • Markers:
    • Their code: <<<<HEAD
    • Separator: ====
    • Your code: >>>>BRANCH_NAME

Rebasing

  • Add commits in a linear fashion (different from the date of committing)
  • Use git rebase TARGET_BRANCH_NAME to merge the current branch's changes with the target
  • Ex: git rebase remote/master to merge with remote's master.
  • For added visualization and interaction in the process you can use git rebase -i TARGET_BRANCH_NAME

Undoing Commits (from safest)

  • git revert creates a new commit that removes all the changes from the given commit
  • git reset
  • git commit-amend changes commit message
  • git cherry-pick

Be careful when changing commit IDs of commits on remote repositories
